# Dreamday Designers App

This is a complete Python Tkinter desktop application for "Dreamday Designers".

## Features
- **Dashboard**: View high-level statistics like active projects and happy clients.
- **Services**: View the list of services offered by Dreamday Designers.
- **New Client Form**: Register a new client inquiry and capture their service interest.

## Requirements
- Python 3.6 or later.
- No external dependencies are required. `tkinter` is included in the standard library.

## How to Run
1. Open a terminal or command prompt.
2. Navigate to the folder containing this code.
3. Run the following command:
   ```bash
   python main.py
   ```
